# Information Memorandum (PDF)

There is no public download page or button. By design, the Information Memorandum is
treated as a **lead-capture asset**: buyers request it via the enquiry form on
`enquire.html` (the "Please send me the Information Memorandum" checkbox), and you
send it to them directly.

If you keep a master PDF in the repo for convenience, place it here, e.g.:

```
assets/docs/serra-luma-information-memorandum.pdf
```

…and attach it in your reply, or wire up an autoresponder in your form service
(Netlify Forms notifications, Zapier, etc.). Nothing on the site links to it
publicly, so the property remains private until you choose to share it.
